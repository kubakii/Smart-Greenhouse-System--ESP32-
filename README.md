This project is the ESP32 side of the intelligent greenhouse management system, including tasks for serial transmission, serial reception, LVGL display, WiFi/MQTT, sensor data processing, and system monitoring.

## 开发环境

### 硬件平台
| 项目 | 规格 |
|:---|:---|
| 主控芯片 | ESP32（乐鑫） |
| 连接方式 | USB 转串口/JTAG |

### 宿主机环境
| 项目 | 配置 |
|:---|:---|
| 操作系统 | Windows（宿主机）+ Linux 虚拟机 |
| 终端工具 | MobaXterm Personal 24.0 |
| 代码编辑 | Visual Studio Code + Remote - SSH 插件 |

### 固件开发框架
| 项目 | 版本 |
|:---|:---|
| SDK | ESP-IDF v5.2 |
| GUI 库 | LVGL v8.3.10 |
